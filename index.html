About us

This website was created in order to integrate the voting system of Indonesia with modern Technologies like website or apps.

Before this website exist many people decide to vote offline because based on the Indonesian Rules about voting Voting must be done openly,common,free,secret,Integrity,and fair. Because of this Indonesia doesn't have a technology like website or application to support it in order to make sure the election is fair and can't be rigged.

The solution is to create a website or app that is secure and can't be hacked by someone in order to seure the people's votes. this is the reason why i created this site, so people can vote whic leader their like anytime and anywhere
